# 标准验收数据集

> **文档版本**：v1.0 | **最后更新**：2026-03-26
> **关联文档**：PRD 10.5（验收质量指标）、商品归组算法说明

------

## 1. 概述

### 1.1 目的

本文档定义 MVP 验收所需的标准测试数据集和预期结果。验收时将系统实际输出与预期结果对比，判断系统是否达到可用水平。

### 1.2 数据集总览

| 数据集 | 场景 | 供应商数 | 文件格式 | 验证重点 |
|--------|------|----------|----------|----------|
| DS-1 | 标准场景 | 3 | 全部 .xlsx | 基础流程闭环、自动映射 |
| DS-2 | 混合格式 + 字段名不一致 | 3 | .xlsx + .docx + PDF | 多格式解析、规则映射、手工修正 |
| DS-3 | 异常场景 | 4 | 全部 .xlsx | 税价/单位/币种异常检测 |
| DS-4 | 归组挑战 | 3 | 全部 .xlsx | 品牌别名、噪音词、归组置信度分层 |
| DS-5 | OCR 路径（可选） | 2 | .xlsx + 扫描版 PDF | 实验性 OCR 路径 |

### 1.3 验收指标

| 指标 | 度量方式 | MVP 目标 |
|------|----------|----------|
| 字段自动映射命中率 | 自动映射正确数 / 总字段数（DS-1、DS-2） | ≥ 80% |
| 高置信归组无需修改率 | 高置信组中用户无需拆分修改的占比（DS-4） | ≥ 90% |
| 异常检出率 | 预设异常被系统正确标记的占比（DS-3） | 100% |
| 追溯完整率 | 导出结果中可追溯到来源的占比 | 100% |
