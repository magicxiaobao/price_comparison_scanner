# UI 手工检查清单

- [ ] 后端已启动：`cd backend && uvicorn main:app --host 127.0.0.1 --port 17396 --reload`

## DS-1 完整流程（标准场景）

### 阶段 1：文件导入

- [ ] 新建项目「DS-1 测试」
- [ ] 导入 3 个 xlsx 文件：
  - `联想科技-办公设备报价.xlsx`
  - `华硕经销-办公设备报价单.xlsx`
  - `戴尔直销-报价函.xlsx`
- [ ] 上传进度条正常显示
- [ ] 解析完成后文件列表显示 3 个文件
- [ ] 供应商名称自动识别正确：
  - 联想科技
  - 华硕经销
  - 戴尔直销
- [ ] 每个文件显示正确的 Sheet 名称（报价明细 / Sheet1 / 报价表）
- [ ] 选择每个供应商的报价表格参与比价

### 阶段 2：标准化

- [ ] 点击执行标准化
- [ ] 异步任务进度提示正常显示
- [ ] 标准化完成后显示映射结果
- [ ] 检查列名映射结果：
  - 供应商 A：品名→商品名称、规格型号→规格型号、单价→单价、金额→总价、交货期→交期
  - 供应商 B：产品名称→商品名称、报价→单价、合计→总价、增值税率→税率、供货周期→交期
  - 供应商 C：商品名称→商品名称、配置→规格型号、含税单价→单价、交期→交期
- [ ] 记录自动映射命中率：__/15 = __% （目标 >= 80%）
- [ ] 需要手动修正的列名能正常修正（如有）
- [ ] 标准化预览表格显示 15 行数据

### 阶段 3：商品归组

- [ ] 点击执行归组
- [ ] 归组完成后显示 5 个组
- [ ] 所有归组为中置信（品牌不同触发硬约束）：
  - 组 1：笔记本电脑（ThinkPad E14 / ASUS VivoBook 14 / Dell Latitude 3440）
  - 组 2：27 寸显示器
  - 组 3：无线键鼠套装
  - 组 4：扩展坞
  - 组 5：激光打印机
- [ ] 归组理由文本可读
- [ ] 手动确认所有归组
- [ ] 确认后归组状态变为「已确认」

### 阶段 4：比价（跳过符合性审查）

- [ ] 点击执行比价
- [ ] 比价完成后显示结果表格
- [ ] 检查比价结果是否正确：
  - 笔记本电脑：最低价 3899（华硕）
  - 27 寸显示器：最低价 1899（联想）
  - 无线键鼠套装：最低价 99（华硕）
  - 扩展坞：最低价 799（华硕）
  - 激光打印机：最低价 1599（戴尔/兄弟）
- [ ] 无异常标记

### 阶段 5：导出

- [ ] 点击导出 Excel
- [ ] 导出任务进度提示正常
- [ ] 导出完成后显示文件路径/下载链接
- [ ] 打开导出的 Excel 文件
- [ ] 检查 Sheet 数量为 3（比价汇总、明细数据、追溯信息）
- [ ] 比价汇总 Sheet 数据与界面一致
- [ ] 追溯信息 Sheet 每行有来源文件名 + Sheet 名 + 行号信息
